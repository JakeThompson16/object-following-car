# Object Following Car
#### Disclaimer
  - Followed build kit intructions for assembly but all code is proprietary (Keyestudio 4WD Bluetooth-compatible Robot Car V2.0 Kit)
## Function
  - RC Car which supports user controlled driving and autonomous object following
## Relevant Hardware Components
  - 4 DC Motors
  - Ultrasonic Sensor mounted to Servo Motor
  - Arduino UNO (Different board than board included in build kit)
  - IR Receiver
  - IR Remote
  - Development Board
  - 8x16 Dot Matrix (not yet functionally implemented via code)
## IR Remote Controls
  - Up -> Drive Straight
  - Left -> Turn Left
  - Right -> Turn Right
  - Down -> Stop
  - 1 -> Slow Speed Mode
  - 2 -> Medium Speed Mode
  - 3 -> Fast Speed Mode
  - Ok -> Toggle Object Following
  - '#' -> Trigger Basic LED Displa Action
